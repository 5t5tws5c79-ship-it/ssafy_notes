# 📄 README (리드미)

## README란?

프로젝트의 **대문(첫 화면)이자 사용설명서**.  
GitHub 저장소에 들어가면 가장 먼저 보이는 문서로, 이 프로젝트가 **무엇인지 · 어떻게 쓰는지**를 알려준다.

- 파일명은 보통 `README.md` (마크다운으로 작성)
- 저장소 최상위에 두면 자동으로 첫 화면에 표시됨

주로 담는 내용: 프로젝트 소개, 설치 방법, 사용법, 기능, 라이선스 등.

---

## README 예시 (마크다운 스니펫)

아래 내용을 복사해서 `README.md`로 저장하면 된다.
