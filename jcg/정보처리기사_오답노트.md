## 🎯 종합 학습 제안

1. **4과목·5과목을 최우선 복습** — 전체 오답 39개 중 21개(53%)가 이 두 과목에 몰려 있음.
2. 반복 출제되는 3대 취약 테마
   - **OS 기본 개념**: 커널 vs 쉘, 프로세스 상태, 스레드, 페이지 교체 알고리즘, 교착상태 해결 기법(예방/회피/발견/복구)
   - **네트워크 계층·프로토콜**: OSI 7계층 vs TCP/IP, 계층별 대표 프로토콜(ARP, RIP, TCP 등)
   - **SW 생명주기·비용산정 모형**: 폭포수/나선형/프로토타입 비교, COCOMO/Putnam 비교
3. **3과목(데이터베이스)**은 SQL 분류(DDL/DML/DCL)와 무결성·정규화 기본 정의부터 표로 정리해서 재학습.
4. 1·2과목은 상대적으로 안정적 — 유지 위주로 가볍게만 복습.
